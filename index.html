자기소개서

안녕하세요, 열려있는 인재 우정입니다.

컴퓨터소프트웨어과를 졸업한 후, 2년간 프론트엔드 및 RPA 개발자로 근무하며
UI/UX 설계부터 자동화 솔루션 개발까지 다양한 프로젝트를 경험했습니다.
프론트엔드 개발을 통해 20% 이상 업무 시간을 단축시키는 사용자 인터페이스를 구현했고,
RPA 프로젝트에서는 단순 반복 업무를 자동화하여 연간 약 100시간 이상의 업무 효율을 개선한 경험이 있습니다.

재직 중, 프론트엔드와 백엔드, RPA를 모두 다루는 통합 개발 환경에서
Spring 기반 백엔드 개발을 자연스럽게 접하게 되었고,
특히 데이터를 DB에 저장하고, 사용자 화면에 출력하는 흐름에 흥미를 느껴
본격적으로 백엔드 개발 역량을 키우기로 결심했습니다.

그 과정에서 Java 기반 백엔드 전문 교육기관인 구디 아카데미의 클라우드 기반 자바 개발자 양성 프로그램을 통해
6개월간 Spring Boot, MyBatis를 활용한 RESTful API 개발과 MySQL 데이터베이스 설계 및 구축, AWS 배포까지 실습하며
현업에 가까운 백엔드 개발 환경을 경험했습니다.

최종 프로젝트에서는 UI/UX 개선을 위한 퍼블리싱 가이드라인을 제공해 팀의 개발 효율을 높이고,
팀원에게 Figma 구조와 HTML 구조를 공유해 협업했습니다.

그 결과, 팀 프로젝트에서 최우수 팀으로 선정되었고,
특히 SELECT 쿼리가 미숙한 팀원에게 JOIN, WHERE 구조를 직접 설명해 기능 완성도를 높였고,
기한 내에 안정적인 시스템을 완성했습니다.

이러한 실무 및 프로젝트 경험을 바탕으로
기업의 성장과 문제 해결에 기여하는 주도적이고 성장 지향적인 백엔드 개발자가 되고자 합니다.
