# 🗂️ 주요 기능
반려동물 용품 쇼핑몰로, 고객 페이지와 관리자 페이지를 기획, 제작한 사이트입니다.

## 🛍 고객 기능  
- 주문  
- 결제  
- 후기 작성  
- 장바구니  

## 🛠 관리자 기능  
- 전체 통계  
- 회원 관리  
- 상품 관리  
- 주문 관리
